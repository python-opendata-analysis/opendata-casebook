## イベント概要

書籍『[Pythonではじめるオープンデータ分析](https://www.kspub.co.jp/book/detail/5412251.html)』（講談社）をテーマに、実際に手を動かしてデータ分析を行うスプリントイベントです。

本イベントは「講義形式」ではなく、チームに分かれて実際にコードを書き、成果物を作成する「スプリント（ハッカソン・集中開発）」形式で実施します。
e-Stat, gBizINFOなど書籍で紹介されているデータやその他の公的統計・オープンデータを活用し、Pythonを使って新たな知見の発掘や可視化などを行います。

※ 当日は書籍を参照しながら進める場面がありますので、持参を推奨します（必須ではありませんが、あると理解が深まります）。

### スプリントとは？

決まった時間内で、集中して開発や作業を行い、成果を出すイベント形式です。
今回はチーム（リーダー＋メンバー）に分かれて、データ分析の課題に取り組みます。

## 募集枠について

本イベントでは、円滑なチーム活動のために以下の2つの枠で募集を行います。

1.  チームリーダー枠
      * 当日のチームのまとめ役です（技術的なプロである必要はありません）。
      * 「こんな分析をしてみたい」「このデータを触ってみたい」というテーマ案がある方を歓迎します。
      * **リーダー枠で申し込む場合、参加者への情報に記載されているリンクにやることを記述してください**
      * やることが決まっていない場合は、「なにかやる」とか適当に書いておいて、後で編集してください
2.  メンバー枠
      * リーダーと共に分析に取り組む参加者です。
